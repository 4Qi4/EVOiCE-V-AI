## ✨ 核心特性

- 🎹 **可视化钢琴卷帘**：拖拽创建音符、设置歌词，所见即所得
- 🇯🇵 **假名自动映射**：直接输入「が」「だ」「ぱ」，自动匹配罗马音音源
- 🎶 **WORLD 声码器合成**：音高自然平滑，内置 DNN 自动颤音
- 💾 **工程自动保存**：每编辑一个音符自动备份，永不丢稿
- 📊 **全局波形预览**：播放/导出后自动绘制整首歌曲波形
- 👥 **多歌手管理**：一键切换歌手，查看音源清单
- ⚡ **轻量无依赖**：Windows 直接运行，无需复杂配置

## 🚀 快速上手

### 1. 准备歌手音源

在 `singers/` 下新建一个歌手文件夹，放入单音素 `.wav`：

- 元音：`a.wav`, `i.wav`, `u.wav`, `e.wav`, `o.wav`
- 辅音：`ka.wav`, `ga.wav`, `da.wav`, `pa.wav`, `shi.wav`, `chi.wav` 等

> 建议音源长度 0.3–1.5 秒，单声道、无背景噪音。

### 2. 启动编辑器

```bash
python evoice.py
```

左侧会自动加载第一个歌手，无需手动选择。

### 3. 三步做一首歌

1. **画音符**：在钢琴卷帘上拖动鼠标创建音符
2. **输歌词**：输入日文假名（如 `が` `だ` `し`），自动匹配音源
3. **点播放**：一键渲染并试听，底部自动显示波形预览

## 🎹 使用说明

### 音符操作

- 拖动：创建/移动音符
- 输入假名：自动映射罗马音音源
- Delete：删除选中音符
- Ctrl+S：保存工程（.evgc）

### 播放与导出

- ▶ 播放：实时渲染并播放合成歌声
- 💾 保存工程：保存 `.evgc` 文件，含音符、音高曲线和歌手信息
- 📤 导出 WAV：输出 44100Hz 32bit 音频文件
- 🎤 歌手信息：查看当前所有歌手及其音源列表

### 自动保存

- 路径：`auto_save/`
- 命名格式：`年-月-日-时-分-工程名.evgc`
- 恢复方式：直接从菜单加载或拖入软件

## ❓ 常见问题

### Q：播放时报「找不到音源」
请确认：
- 音源文件放在 `singers/[歌手名]/` 目录下
- 文件名与假名对应的罗马音一致（如 `が` → `ga.wav`）

### Q：声音沙哑/不自然
- 音源尽量使用**干净、无杂音、无尾音的单音素**
- 长度控制在 0.3–1.5 秒，避免过长尾音

### Q：工程丢了怎么办
- 所有编辑都会自动保存到 `auto_save/` 目录
- 直接加载最新的 `.evgc` 文件即可恢复
