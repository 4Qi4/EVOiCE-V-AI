# EVOiCE - 简易歌声编辑器

一个轻量级的UTAU类歌声编辑器，支持单音素材库、音高拉伸、钢琴卷帘界面和AI自动音高功能。

## 功能特点

### 🎵 核心功能
- **单音素材库**：支持导入和管理WAV格式的单字音源
- **音高拉伸与变调**：使用PYWORLD进行高质量的音高转换
- **钢琴卷帘界面**：可视化编辑音符、歌词和音高曲线
- **AI自动音高**：内置手搓版AI算法，自动生成自然的音高曲线

### 🎨 界面特色
- **深色主题**：护眼的深色界面设计
- **实时预览**：边编辑边听效果
- **直观操作**：点击创建音符，拖拽调整时长和音高
- **参数面板**：调整歌词、力度、颤音等参数

### 🚀 技术亮点
- **纯Python实现**：无需复杂的编译环境
- **手搓AI算法**：自相关法音高检测 + 自动滑音/颤音生成
- **模块化设计**：易于扩展和定制
- **跨平台兼容**：支持Windows、macOS、Linux

## 使用教程

### 1. 启动软件
```bash
python evoice.py
```

### 2. 创建新项目
- 点击"新建"按钮或按 `Ctrl+N`
- 在钢琴卷帘上点击创建音符
- 双击音符可以编辑歌词

### 3. 导入音源
- 点击"选择"按钮选择音源文件夹
- 音源文件命名格式：`c4.wav`, `d#5.wav` 等

### 4. 编辑音符
- **创建音符**：在钢琴卷帘上点击
- **调整音高**：拖拽音符上下移动
- **调整时长**：拖拽音符左右边缘
- **编辑歌词**：在右侧参数面板输入或双击音符

### 5. AI自动音高
- 点击"AI自动音高"按钮生成自然音高曲线
- 可调整颤音速率和深度参数
- 音高曲线会自动添加滑音和颤音效果

### 6. 播放与导出
- 点击"播放"按钮或按空格键预览效果
- 点击"导出音频"保存为WAV文件

## 快捷键

- `空格键`：播放/暂停
- `Delete`：删除选中音符
- `Ctrl+S`：保存项目
- `Ctrl+O`：打开项目
- `Ctrl+N`：新建项目

## 音源制作

### 创建自己的音源
1. 录制单个音高的干声（WAV格式）
2. 文件名格式：`音符名+八度.wav`（如 `c4.wav`, `a3.wav`）
3. 将所有音源文件放在同一个文件夹中
4. 在软件中选择该文件夹作为音源目录

### 推荐录音设置
- 采样率：44.1kHz
- 位深度：16bit
- 格式：WAV（无损）
- 时长：1-2秒的稳定发音

## 技术原理

### 音高转换
使用PYWORLD声码器算法，保持音色不变的情况下转换音高。

### AI自动音高算法
1. **自相关法音高检测**：手动实现的音高提取算法
2. **自动滑音生成**：音符之间的平滑过渡
3. **自动颤音添加**：模拟人声的自然颤音效果

### 音频处理流程
1. 读取音源文件
2. 音高转换（根据音符MIDI值）
3. 时间拉伸（根据音符时长）
4. 应用力度和包络
5. 拼接所有音符
6. 导出最终音频

## 常见问题

### Q: 为什么没有声音？
A: 请检查是否正确选择了音源文件夹，并且文件夹中有对应的WAV文件。

### Q: 音高转换效果不好怎么办？
A: 尝试使用更高质量的音源文件，确保录音环境安静，发音稳定。

### Q: 可以导入现有UTAU音源吗？
A: 可以，将UTAU音源的WAV文件复制到一个文件夹中，在软件中选择该文件夹即可。

### Q: 支持中文歌词吗？
A: 完全支持，直接在歌词输入框中输入中文即可。
